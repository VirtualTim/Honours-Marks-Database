                    <div class="highlight"><pre><div class='line' id='LC1'>This database schema defines a databse to hold marks for an honurs course.</div><div class='line' id='LC2'><br/></div><div class='line' id='LC3'>The &quot;Staff&quot; table:</div><div class='line' id='LC4'>It is a table to hold information about markers and supervisors.</div><div class='line' id='LC5'>StaffID holds a unique identifier for each staff member; either their UWA staff number, or if they</div><div class='line' id='LC6'>do not have one, a uniquely generated number.</div><div class='line' id='LC7'>FirstName and LastName hold the first and last name of the staff member.</div><div class='line' id='LC8'><br/></div><div class='line' id='LC9'>The &quot;Student&quot; table:</div><div class='line' id='LC10'>It is a table to hold information about students.</div><div class='line' id='LC11'>StudentID holds the student&#39;s unique student number.</div><div class='line' id='LC12'>FirstName and LastName hold the first and last name of the student.</div><div class='line' id='LC13'>DissTitle is the title of that student&#39;s dissertation.</div><div class='line' id='LC14'>Discipline is the discipline being studied by that student; one of ANHB, PHYL, NEURO, or BIOMS</div><div class='line' id='LC15'>Supervisors1-4 are the StaffIDs corresponding to that student&#39;s supervisor</div><div class='line' id='LC16'><br/></div><div class='line' id='LC17'>The &quot;Unit&quot; table:</div><div class='line' id='LC18'>It is a table to hold information about units.</div><div class='line' id='LC19'>UnitCode is the unique 4 letter, 4 number code for the unit.</div><div class='line' id='LC20'>UnitName is the name of the unit.</div><div class='line' id='LC21'>Points is the number of points the unit is worth (ie. 6 or 12).</div><div class='line' id='LC22'><br/></div><div class='line' id='LC23'>The &quot;Discipline&quot; table:</div><div class='line' id='LC24'>It is a table to hold information about disciplines.</div><div class='line' id='LC25'>DisciplineName is the name of the discipline, corresponding to the Discipline field in the Student table</div><div class='line' id='LC26'>UnitCode1-4 are the units that are studied under the discipline, corresponding to the UnitCode in the Unit table.</div><div class='line' id='LC27'><br/></div><div class='line' id='LC28'>The &quot;Assessment&quot; table:</div><div class='line' id='LC29'>It is a table to hold information about assessments.</div><div class='line' id='LC30'>AssessmentID is a uniquely generated integer to refer to each assessment.</div><div class='line' id='LC31'>AssessmentName is the name of the assessment.</div><div class='line' id='LC32'>UnitCode is the UnitCode of the unit for which this assessment was set.</div><div class='line' id='LC33'>UnitPercent is the percentage of that unit that this assessment makes up.</div><div class='line' id='LC34'><br/></div><div class='line' id='LC35'>The &quot;SubAssessment&quot; table:</div><div class='line' id='LC36'>It is a table to hold information about subassessments.</div><div class='line' id='LC37'>SubAssessmentID is a uniquely generated integer to refer to each subassessment.</div><div class='line' id='LC38'>SubAssessmentName is the name of the subassessment.</div><div class='line' id='LC39'>AssessmentID is the AssessmentID for the assessment that this subassessment is a part of.</div><div class='line' id='LC40'>AssessmentPercent is the percentage of that assessment that this subassessment makes up.</div><div class='line' id='LC41'>MaxMarks is the number of marks this subassessment was marked out of.</div><div class='line' id='LC42'><br/></div><div class='line' id='LC43'>The &quot;SubAssessmentMarks&quot; table:</div><div class='line' id='LC44'>It is a table to hold information about subassessment marks.</div><div class='line' id='LC45'>MarkID is a uniquely generated integer to refer to each subassessment mark.</div><div class='line' id='LC46'>Mark is the integer value of the mark given.</div><div class='line' id='LC47'>MarkerID is the StaffID of the marker that gave the mark.</div><div class='line' id='LC48'>StudentID is the StudentID of the student that recieved this mark.</div><div class='line' id='LC49'>SubAssessmentID is the subassessment that this mark was given for.</div><div class='line' id='LC50'>InsideRange is an integer to be set to 0 or 1, to tell if the mark is or isn&#39;t within 2 standard deviations</div><div class='line' id='LC51'>of the average for this student; 0 if it is, 1 if it isn&#39;t. If it is outside this range, it is a flag to</div><div class='line' id='LC52'>disregard the mark.</div><div class='line' id='LC53'><br/></div><div class='line' id='LC54'>The &quot;AssessmentMarks&quot; table:</div><div class='line' id='LC55'>It is a table to hold information about assessment marks.</div><div class='line' id='LC56'>MarkID is a uniquely generated integer to refer to each assessment mark.</div><div class='line' id='LC57'>Mark is the integer value of the mark given.</div><div class='line' id='LC58'>StudentID is the StudentID of the student that recieved this mark.</div><div class='line' id='LC59'>AssessmentID is the assessment that this mark was given for.</div><div class='line' id='LC60'><br/></div><div class='line' id='LC61'>The &quot;UnitMarks&quot; table:</div><div class='line' id='LC62'>It is a table to hold information about whole unit marks.</div><div class='line' id='LC63'>MarkID is a uniquely generated integer to refer to each unit mark.</div><div class='line' id='LC64'>Mark is the integer value of the mark given.</div><div class='line' id='LC65'>Grade is the corresponding grade for that mark (ie. C for 60-69, D for 70-79, etc.)</div><div class='line' id='LC66'>StudentID is the StudentID of the student that recieved this mark.</div><div class='line' id='LC67'>UnitCode is the unit that this mark was given for.</div><div class='line' id='LC68'><br/></div><div class='line' id='LC69'>The &quot;Information&quot; table&quot;</div><div class='line' id='LC70'>It is a table to hold information about the database itself.</div><div class='line' id='LC71'>Lock is a variable, defaulting to &#39;X&#39;, that is held to ensure only one row is kept in the &#39;Information&#39; table.</div><div class='line' id='LC72'>Period holds the semester and year (ie. 20131 for semester 1 2013) that the database is relevant to. It is</div><div class='line' id='LC73'>also the name of the database, for archiving purposes.</div><div class='line' id='LC74'>LastBackup holds the timestamp that the last backup was saved.</div><div class='line' id='LC75'>AutoBackupSchedule holds 1 character to represent the current backup scheduel;&#39;n&#39; for never, &#39;d&#39; for daily,</div><div class='line' id='LC76'>or &#39;w&#39; for weekly.</div><div class='line' id='LC77'>AutoBackupDue holds the timestamp at which the next autobackup is set to occur.</div><div class='line' id='LC78'>LastRestoreDate holds the timestamp for the last time a backup was used to restore the database.</div><div class='line' id='LC79'>LastRestoreFrom holds the name of the backup database that was last used to restore the database.</div><div class='line' id='LC80'>VersionSuperseded is the name of a copy of the database that was replaced the last time a restore occurred.</div></pre></div>
